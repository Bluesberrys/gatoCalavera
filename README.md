# Gato Calavera

## Description

Gato Calavera is a web system for a clothing brand with a marketplace. The brand focuses on designing casual wear for the Mexican Furry community.

## Features

- Responsive design
- Product showcase
- Contact form
- About us page

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express
- Body-parser
- Cors
- Morgan
- Dotenv
